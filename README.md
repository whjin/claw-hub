## 核心特性

1. 完整openClaw分层架构：网关/Agent核心层/多模型适配层/技能层/记忆层
2. 原生支持国内主流大模型：智谱AI、通义千问、豆包、Kimi、DeepSeek、腾讯混元等
3. 工厂模式设计：新增模型仅需加配置，无需修改核心代码
4. 内置工具调用能力：可无限扩展自定义技能
5. 会话记忆持久化：重启服务不丢失对话历史
6. 开箱即用的Web交互界面：无需额外搭建前端项目

## 开发环境

1. Node.js 22+
2. pnpm包管理器
3. 大模型API Key
